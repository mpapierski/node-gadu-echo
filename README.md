# node-gadu-echo

A simple example of the [node-gadu](http://github.com/mpapierski/node-gadu) library.

## Usage

	export UIN=1234 PASSWORD=password
	cd ~/node-gadu-echo
	npm install
	node app
	
Send a message to `1234` and see what happens.

## Requirements

	node >= 0.8
	libgadu
